# Chrome Log Collector

A simple Chrome extension that provides users with an easy way to download a diagnostic tool for collecting detailed Chrome browser logs.

## Overview

The Chrome Log Collector extension helps users troubleshoot web application issues by providing a convenient way to download a specialized logging tool. This tool captures comprehensive Chrome browser logs that can be shared with technical support teams for analysis.

## Features

- Simple, user-friendly interface
- One-click download of the Log Collector utility
- Clear instructions for users
- No installation required for the tool itself
- Collects detailed Chrome logs not accessible through standard methods

## How It Works

1. User clicks the Chrome Log Collector extension icon
2. Clicks "DOWNLOAD LOG COLLECTOR" in the popup
3. Runs the downloaded executable
4. Starts logging and reproduces the issue in Chrome
5. Stops logging to collect and save logs to desktop
6. Shares logs with technical support

## Extension Structure

- `manifest.json` - Extension configuration
- `popup.html` - User interface
- `popup.js` - Download functionality
- `images/` - Extension icons

## Privacy

This extension:
- Does not collect, store, or transmit any personal data
- Only uses the downloads permission to download the logging tool
- Does not automatically send any data to third parties

Users maintain full control over the log files generated by the tool, which are only shared with technical support if the user chooses to do so.

## Installation

### For Users
1. Install from the Chrome Web Store (link coming soon)
2. Click the extension icon when you need to collect logs

### For Developers
1. Clone this repository
2. Open Chrome and go to `chrome://extensions/`
3. Enable "Developer mode"
4. Click "Load unpacked" and select the `Log_Collector` folder
5. The extension will appear in your extensions list

## Distribution

To distribute this extension:

1. Host the executable file (Log-Collector.exe) on a reliable hosting service
2. Update the `executableUrl` in `popup.js` with the actual download URL
3. Package the extension for the Chrome Web Store
